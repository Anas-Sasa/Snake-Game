# Snake-game

A classic Snake game built with Python's Turtle module.

## Project Structure:
> - `main.py` - Main game logic and loop and collision detection
> - `snake.py` - Snake class handling movement, growth and control over in general
> - `food.py` - Food generation and hide it 
> - `scoreboard.py` - Score tracking and display

## How to Play:

> - Use arrow keys to control the snake
> - Eat food to grow and increase score
> - Avoid hitting walls or yourself
